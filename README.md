# Kipu Grapher

Telegram bot to graph friday bottle raffle progress.

## Getting Started

### Basic

1. Set `BOT_TOKEN` in .env
2. pipenv install --dev
3. pipenv shell
4. python3 setup.py
5. watchgod bot.main

### Docker

1. docker build -t [image_name] .
2. docker run -e "BOT_TOKEN=???" [image_name]
